Dictation (Speech Recognition) is the most popular and easiest way to Write a POST with your voice, converting voice on text with any language.

== Description ==
Dictation (Speech Recognition) adds an interactive MIC icon to the wordpress editing screens.  When editing a Post or Page just click on MIC icon and start talking.

The plugin will automatically insert the converted text from voice into your blog!

= Key Features =
* Dictation (Speech Recognition) is based on the latest Google API
* WordPress MultiSite compatible
* Custom post types are supported
* Easily use your voice in the standard post edit and page edit screens
* Real-time voice to text converter using Chrome on Desktop
